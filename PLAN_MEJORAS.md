# Plan de Mejoras: Recursos y Scripting Lab

## Resumen del Proyecto
Aplicación HOI4 Mod Studio para crear mods de Hearts of Iron IV.

## Mods Base Soportados
- Vanilla / Historical
- Millennium Dawn
- Kaiserreich
- The New Order (TNO)
- Road to 56

## Estado Actual

### Recursos (IconTool, FlagTool, PortraitTool)
- Disponible para todos los mods base
- Permite subir y convertir imágenes
- NO muestra el mod base actualmente seleccionado

### ScriptingLab
- Disponible para todos los mods base
- Diccionario de comandos Clausewitz
- NO se adapta según el mod base

## Mejoras Propuestas

### 1. Mostrar Mod Base Actual
Agregar badge con el nombre del mod seleccionado en cada editor.

### 2. Adaptar IconTool por Mod Base
Categorías de assets específicas según el mod.

### 3. Adaptar ScriptingLab por Mod Base
Diccionario de comandos adaptable según el mod.

### 4. Cambio de Tema Visual (Opcional)
Cambiar colores según el mod base.
